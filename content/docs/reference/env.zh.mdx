## 概述

环境变量从 `.env`（生产环境）和 `.env.local`（本地覆盖，不提交 Git）加载。

## 核心配置

| 变量 | 默认值 | 必需 | 描述 |
|----------|---------|----------|-------------|
| `JWT_SECRET` | `change-me-in-production` | ✅ | JWT 签名密钥（生产环境必须修改） |
| `ENCRYPTION_KEY` | (回退到 JWT_SECRET) | | AES-256-GCM 加密密钥 |
| `PORT` | `41905` | | HTTP 服务器端口 |
| `DATABASE_URL` | `file:./data/aigcs.db` | | 数据库连接 URL |
| `NODE_ENV` | `production` | | 运行环境 |
| `ADMIN_URL` | `http://localhost:5173` | | 管理面板 URL（用于邮件模板） |
| `ALLOWED_ORIGINS` | `[]` | | CORS 允许的来源（JSON 数组字符串） |
| `REGISTRATION_OPEN` | `true` | | 允许公开注册 |
| `RATE_LIMIT_MAX` | `100` | | 每个窗口最大请求数 |
| `RATE_LIMIT_WINDOW` | `60` | | 速率限制窗口（秒） |
| `GLOBAL_SYSTEM_PROMPT` | — | | 全局 AI 系统提示 |
| `PLUGINS_DIR` | (自动检测) | | 覆盖插件目录路径 |

## SMTP

| 变量 | 描述 |
|----------|-------------|
| `SMTP_HOST` | SMTP 服务器主机名 |
| `SMTP_PORT` | SMTP 端口 |
| `SMTP_USER` | SMTP 用户名 |
| `SMTP_PASS` | SMTP 密码 |
| `SMTP_FROM_EMAIL` | 发件人邮箱地址 |
| `SMTP_FROM_NAME` | 发件人显示名称 |
