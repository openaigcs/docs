## 概述

所有 API 端点返回 JSON 响应。

### 基础 URL

```
http://您的服务器:41905/api
```

### 响应格式

```json
{
  "code": 200,
  "data": { ... }
}
```

错误响应：

```json
{
  "code": 401,
  "message": "Unauthorized"
}
```

### 身份验证

| 方式 | 请求头 |
|--------|--------|
| JWT | `Authorization: Bearer <token>` |
| API Token | `Authorization: Bearer <token>` |
| CSRF（管理端） | `X-Requested-With: XMLHttpRequest` |

## 认证端点（`/api/auth`）

| 方法 | 路径 | 认证 | 描述 |
|--------|------|------|-------------|
| POST | `/register` | — | 注册新用户 |
| POST | `/login` | — | 通过邮箱或用户名登录 |
| POST | `/totp/verify` | — | 完成 TOTP 升级认证 |
| GET | `/me` | Bearer | 获取当前用户资料 |
| PUT | `/me` | Bearer | 更新资料 |
| POST | `/change-password` | Bearer | 修改密码 |
| POST | `/refresh` | — | 刷新访问令牌 |
| POST | `/avatar` | Bearer | 上传头像（最大 2MB） |
| DELETE | `/avatar` | Bearer | 移除头像 |
| GET | `/avatar/:userId` | — | 提供头像图片 |
| POST | `/totp/setup` | Bearer | 生成 TOTP 密钥 + QR 码 |
| POST | `/totp/enable` | Bearer | 启用 TOTP |
| POST | `/totp/disable` | Bearer | 禁用 TOTP |

## 管理端点（`/api/admin`）

所有请求需要 `Authorization: Bearer <token>` + `X-Requested-With: XMLHttpRequest`。

|--------|------|-------------|
| GET | `/sites` | 列出用户站点 |

| PUT | `/sites/:id` | 更新站点 |
