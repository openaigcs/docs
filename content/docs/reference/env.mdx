## Overview

Environment variables are loaded from `.env` (production) and `.env.local` (local overrides, not committed).

## Core Configuration

| Variable | Default | Required | Description |
|----------|---------|----------|-------------|
| `JWT_SECRET` | `change-me-in-production` | ✅ | JWT signing key (MUST change in production) |
| `ENCRYPTION_KEY` | (falls back to JWT_SECRET) | | AES-256-GCM encryption key |
| `PORT` | `41905` | | HTTP server port |
| `DATABASE_URL` | `file:./data/aigcs.db` | | Database connection URL |
| `NODE_ENV` | `production` | | Runtime environment |
| `ADMIN_URL` | `http://localhost:5173` | | Admin panel URL (for email templates) |
| `ALLOWED_ORIGINS` | `[]` | | CORS allowed origins (JSON array string) |
| `REGISTRATION_OPEN` | `true` | | Allow public registration |
| `RATE_LIMIT_MAX` | `100` | | Max requests per window |
| `RATE_LIMIT_WINDOW` | `60` | | Rate limit window in seconds |
| `GLOBAL_SYSTEM_PROMPT` | — | | Global AI system prompt |
| `PLUGINS_DIR` | (auto-detected) | | Override plugins directory path |

## SMTP

| Variable | Description |
|----------|-------------|
| `SMTP_HOST` | SMTP server hostname |
| `SMTP_PORT` | SMTP port |
| `SMTP_USER` | SMTP username |
| `SMTP_PASS` | SMTP password |
| `SMTP_FROM_EMAIL` | Sender email address |
| `SMTP_FROM_NAME` | Sender display name |
