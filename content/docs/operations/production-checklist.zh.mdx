---

## 必需

- [ ] **更改 JWT_SECRET** — 生成一个随机的 64 字符字符串
- [ ] **设置 ENCRYPTION_KEY** — 与 JWT_SECRET 分开，随机 64 字符
- [ ] **启用 HTTPS** — 使用 Nginx/Caddy 配合 Let's Encrypt
- [ ] **配置 CORS** — 将 `ALLOWED_ORIGINS` 设置为您的站点域名
- [ ] **设置强管理员密码** — 使用密码管理器

## 推荐

- [ ] **配置 SMTP** — 用于邮件通知和密码恢复
- [ ] **启用 CAPTCHA** — Cloudflare Turnstile（免费，无感验证）
- [ ] **启用 TOTP 双因素认证** — 用于管理员账户
- [ ] **设置监控** — 健康检查端点 `/api/health`
- [ ] **定期备份** — 安排每日数据库备份
- [ ] **速率限制** — 根据流量调整限制
- [ ] **数据库连接池** — 根据您的数据库调优

## 可选

- [ ] **自定义域名** — 使用您的域名配置反向代理
- [ ] **CDN 缓存** — Cloudflare 或类似服务用于 widget.js
- [ ] **Redis 缓存** — 用于在高负载下提升性能
- [ ] **PM2 / systemd** — 进程管理器确保运行时间
- [ ] **日志聚合** — 将 Docker 日志转发到外部服务
