## 概述

根据来源系统的不同，从其他评论系统迁移到 AIGCS 只需几个步骤。

## 通用流程

1. **部署 AIGCS** — 参考[快速开始](../getting-started/quick-start)
2. **创建站点** — 匹配您的域名
3. **配置 AI 提供商** — 满足评论需求
4. **嵌入 Widget** — 在您的站点上
5. **重定向旧评论 URL**（如果需要）
6. **停用旧系统** — 确认满意后

## 从 twikoo / waline / artalk 迁移

<Callout type="warn">暂无计划开发针对这些系统的自动迁移工具。</Callout>

要迁移您已有的评论：
1. 从来源系统导出评论备份（通常为 JSON/CSV）
2. 在站点仪表盘 → 评论导入/导出中点击**下载模板**，获取 AIGCS 的评论备份格式
3. 使用 AI 编写转换脚本，将导出的评论备份转换为 AIGCS 的模板格式
4. 通过管理面板的**评论导入/导出**功能导入转换后的文件
