## 概述

根据来源系统的不同，从其他评论系统迁移到 AIGCS 只需几个步骤。

## 通用流程

1. **部署 AIGCS** — 参考[快速开始](../getting-started/quick-start)
2. **创建站点** — 匹配您的域名
3. **配置 AI 提供商** — 满足评论需求
4. **嵌入 Widget** — 在您的站点上
5. **重定向旧评论 URL**（如果需要）
6. **停用旧系统** — 确认满意后

## 从 twikoo / waline / artalk 迁移 [WIP]

<Callout type="warn">从这些系统的自动导入功能已计划但尚未实现。</Callout>

这些系统将访客评论存储在其数据库中。
AIGCS 目前还没有针对这些系统的自动导入工具。

**手动迁移：**
1. 从来源系统导出评论（通常为 JSON/CSV）
2. 使用 AIGCS API 导入访客评论
3. 访客评论导入端点：`POST /api/admin/sites/:siteId/comments/visitor/import`
